# MRT Pass
 
### Dhaka Metro Rail Card Manager for Android
 
*Instantly check your MRT Pass balance, track journey history, and calculate fares — all from your phone.*
 
</div>
 
 
## What is MRT Pass?
 
**MRT Pass** is an Android app that reads your Dhaka Metro Rail (MRT Line 6) contactless card using NFC. No internet required. No account needed. Just tap your card and see your balance.
 
> ⚠️ **Disclaimer:** This is an unofficial project. It is not affiliated with DTCA (Dhaka Transport Co-ordination Authority) or DMTCL (Dhaka Mass Transit Company Limited).

## Features
 
### NFC Card Reading
- Tap your MRT Pass card to your phone — balance appears instantly
- Reads **up to 20 transaction blocks** directly from the card chip
- Uses **Felica / NfcF (ISO 18092)** protocol
- Works fully **offline** — no internet connection required
- Automatically merges new transactions without creating duplicates
 
### Transaction History
- View complete journey history stored on the card
- See **From station → To station** for each trip
- Timestamps with date and time
- MRT Pass fare shown per trip (with 10% discount applied)
- Balance after each transaction
- Trip and Recharge entries shown separately
 
### QR Code — Share & Import
- Generate a QR code containing your card's full data (balance + all transactions)
- Scan QR from another device to import the card instantly
- Share QR as an image via any app
- Useful for devices without NFC
 
### Fare Calculator
- Official DTCA fare matrix for all **17 stations** (17×17 table)
- Shows both **Regular fare** (token) and **MRT Pass fare** (10% discount)
- See how much you save per journey
- Swap FROM/TO stations with one tap
- Visual metro line with animated train icon at selected station
- Tap any station in the list to set it as destination
 
### Multi-Card Support
- Save and manage **multiple MRT Pass cards**
- 3D swipeable card carousel on home screen
- Rename cards with custom nicknames
- Choose from **5 color themes** per card
- Delete cards you no longer need
 
### Analytics
- Total trips across all cards
- Trips this month
- Per-card statistics
- Mini bar chart showing fare amounts for last 7 journeys
 
### Customization
- **Dark / Light / System** theme
- **Bangla & English** language support — switch anytime
- Auto-save toggle after NFC scan
- Animated balance counter with smooth transition
 
## Requirements
 
- Android 6.0 (API 23) or higher
- **NFC-enabled device** (for card reading)
- Camera permission (for QR scanning only)
